# CalcE - 化工工程师个人生产力工具

CalcE 是一套基于 **PySide6** 开发的桌面应用，面向化工工程师，集成了 38+ 专业工程计算器与日常生产力工具。

---

## 🌟 功能模块

### 🔬 工程计算（38+ 计算器）

| 类别 | 计算器 |
|------|--------|
| **管道** | 管径计算、管道压降、管道壁厚、管道跨距、管道间距、管道补偿 |
| **流体设备** | 离心泵功率、NPSHa 汽蚀余量、风机功率 |
| **换热** | 换热器面积、换热器计算、隔热层厚度、蒸汽管径流量、长距离蒸汽管 |
| **热工/制冷** | 制冷循环、蒸汽性质、气体状态转换、湿空气 |
| **容器/结构** | 容器体积计算、罐体重量、篮式过滤器设计 |
| **安全/消防** | 安全阀计算、安全泄放面积、消火栓计算、危化品查询、腐蚀数据查询 |
| **热力学** | 状态方程(EOS)、纯物质性质、气体混合物性质、固体溶解度、汽液平衡(VLE)、混合液闪点 |
| **其他** | 法兰尺寸、压力管道定义、压力管道压缩流压降、制冷剂性质 |

### 📐 单位换算器
支持长度、重量、温度、压力、体积、面积、能量、功率、速度、力等多类换算。

### 🍅 番茄时钟
标准番茄工作法（25+5分钟），自定义时长，支持休息提醒。

### ✅ 待办事项
任务增删改，优先级标记，完成状态跟踪。

### 📝 笔记
分文件夹管理笔记，支持搜索筛选。

### 🔖 书签
网页链接管理，分类整理，快速访问。

### 📅 重要日期
生日、节假日、纪念日管理，倒计时显示。

### ⏰ 倒计时
多事件倒计时，自定义名称和目标时间，实时更新。

### 📊 今年余额
年度时间进度可视化，百分比展示已过/剩余时间。

---

## 🛠️ 技术栈

- **GUI 框架**：Python 3.8+ + PySide6 6.5+
- **科学计算**：NumPy、SciPy
- **PDF 导出**：ReportLab（计算书生成）
- **日志**：Loguru
- **数据存储**：本地 JSON 文件（`~/.calce/` 或 AppData）
- **计算基础**：流体力学、制冷热力学、IAPWS-IF97 水蒸气公式、离心泵理论等

## 🚀 安装与运行

### 环境要求
- Python 3.8+
- Windows 10/11（主要测试平台）

### 安装步骤

```bash
# 1. 克隆仓库
git clone https://github.com/virmuran/CalcE.git
cd CalcE

# 2. 安装依赖
pip install -r requirements.txt

# 3. 运行
python main.py
```

## 📖 使用说明

1. 启动后通过顶部标签页切换各功能模块
2. **工程计算**：选择计算类别 → 填写参数 → 点击「计算」查看结果；压降计算支持导出 PDF 计算书
3. **主题切换**：菜单栏「主题」→ 亮色 / 暗色 / 蓝色
4. **数据备份**：菜单栏「文件」→「备份数据」，备份至 AppData 目录
